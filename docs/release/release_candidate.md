# Release Candidate v3.3

## Overview
FiduScan v3.3 Release Candidate encapsulates all backend capabilities (Deepfake Detection, Watermarking, Authenticity Reports, Evidence Chains, Enterprise Cases, and Blockchain) alongside the fully rewritten Next.js v3.0 forensic frontend.

## Deployment Target
- **Frontend:** Vercel (Next.js)
- **Backend:** Railway (FastAPI Python)
- **Database:** PostgreSQL (Railway managed)
- **Billing:** Stripe (Production Environment)

## Scope
This candidate is targeted specifically for the Closed Beta cohort. It is structurally identical to the proposed General Availability (GA) release, barring the Blockchain integration which remains pointing to a mock/testnet RPC for safety during beta.
