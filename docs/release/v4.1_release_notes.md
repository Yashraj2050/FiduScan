# v4.1 Feedback Release Notes

## Quick Wins (10)
- Improved empty state illustrations in Evidence Vault.
- Added tooltips to the deepfake verification heatmap.
- Adjusted contrast on dark mode buttons for accessibility.
- Optimized report generation endpoint (now < 3s).
- Standardized timestamp formatting across all investigation views.
- Fixed pagination limits in the Case Management table.
- Added copy-to-clipboard functionality for blockchain hashes.
- Reduced initial bundle size of the V3.0 React application.
- Clarified error messaging for unsupported file uploads.
- Streamlined account creation flow by removing an unnecessary step.

## Medium Effort (5)
- **Audio Scrubber:** Implemented a zoomable scrubber in the Audio Evidence Viewer.
- **Bulk Case Export:** Analysts can now select multiple cases and download a single ZIP.
- **Advanced Case Filtering:** Users can filter cases by tags and priority.
- **Billing Tier Visibility:** Clearer breakdown of quotas remaining on the dashboard.
- **Asynchronous Anchoring:** Sped up Evidence Chain UI by pushing Polygon anchoring to a background task.

## Major Changes (2)
- **Granular RBAC:** Expanded the role system based on enterprise feedback to include 'Viewer-Only' and 'Auditor' roles.
- **Webhooks v2:** Upgraded the developer portal with robust webhook signing and delivery retries.
