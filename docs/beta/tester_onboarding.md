# Tester Onboarding Flow

## 1. Account Creation
- Testers receive a magic link via email inviting them to the `fiduscan-beta` workspace.
- Account is automatically provisioned with 'Analyst' RBAC role and a high-tier beta quota.

## 2. First Investigation
- User logs into the Dashboard.
- Navigates to **Investigations -> Create New Case**.
- Uploads sample media (provided in the welcome packet).

## 3. First Report & Evidence Chain
- User triggers a forensic scan on the uploaded media.
- Verifies the automatic generation of the JSON/PDF report.
- Views the Evidence Chain to see the initial timestamp and provenance logs.

## 4. First Export
- User navigates back to the active Case.
- Clicks "Export Package" to download a ZIP containing the media, report, and audit trail.
