# Tester Checklist

Please complete the following actions during your beta testing period. If you encounter friction or bugs, please log them using the bug reporting workflow.

- [ ] Log in using the provided magic link.
- [ ] Upload an image, audio file, or video to the Deepfake Scanner.
- [ ] View the Authenticity Score and Heatmap/Spectrogram.
- [ ] Create a New Case in the Investigations tab.
- [ ] Link your uploaded media to the Case as Evidence.
- [ ] Embed a FiduScan watermark into a clean piece of media.
- [ ] Extract the watermark using the Verification tab.
- [ ] Add an Investigator Note to your Case summarizing your findings.
- [ ] Download the generated Authenticity Report (PDF or JSON).
- [ ] Click "Export Package" on your Case to download the full ZIP bundle.
- [ ] Validate the blockchain anchor hash on the Polygon block explorer (using the link in the Evidence tab).
