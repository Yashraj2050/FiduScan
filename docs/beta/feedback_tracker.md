# Beta Feedback Tracker (Master)

This document serves as the master aggregation board for all beta feedback across 8 primary categories. Specific bugs and feature requests are detailed in their respective sub-trackers.

## Categories Monitored
1. **Bugs:** Critical, High, and Minor technical failures. *(See `bug_tracker.md`)*
2. **UX Friction:** Confusing UI patterns, poor contrast, or un-intuitive flows.
3. **Feature Requests:** Net-new functionality requested by testers. *(See `feature_request_tracker.md`)*
4. **False Positives:** Authentic media incorrectly flagged as AI.
5. **False Negatives:** AI media incorrectly flagged as Authentic.
6. **Performance Issues:** Latency, UI stuttering, or slow API responses.
7. **Billing Issues:** Unexpected subscription limits or Stripe integration failures.
8. **API Issues:** Authentication failures, rate limit friction, or documentation gaps.
