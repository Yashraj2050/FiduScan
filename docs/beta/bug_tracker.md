## Triage Severities
- **P0 (Blocker):** System crash, data loss, or core inference failure. (SLA: < 1 hour)
- **P1 (High):** Major feature broken (e.g., cannot upload video). (SLA: < 4 hours)
- **P2 (Medium):** UI glitch, minor integration bug. (SLA: Next Patch)
- **P3 (Low):** Typo, visual alignment. (SLA: Backlog)
