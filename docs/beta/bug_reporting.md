# Bug & Feature Reporting Workflow

## Reporting a Bug
If you encounter an error, unexpected behavior, or a crash, please report it via the in-app "Give Feedback" widget or in the Slack channel.

**Please include:**
1. **Description:** What happened?
2. **Steps to Reproduce:** How did you trigger the bug?
3. **Expected Behavior:** What should have happened?
4. **Environment:** Browser (Chrome, Safari, etc.) and OS.
5. **Assets:** If an image or video caused a failure, please attach the file (or a hash if confidential).

## Requesting a Feature
If a workflow feels incomplete or you need additional capabilities for your use case, use the Slack channel.

**Please include:**
1. **The Goal:** What are you trying to accomplish?
2. **The Friction:** Why is the current platform preventing you from doing it easily?
3. **The Suggestion:** How would you like it to work?
