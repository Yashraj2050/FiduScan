# Session Recovery Guide

If you lose context or are restarted, execute the following to recover state:
1. Read `docs/context/project_checkpoint_v6.md` for architecture details.
2. Read `docs/context/resume_prompt.md` to understand your current persona and goal.
3. Check `docs/context/pause_state.md` for the exact phase you are currently executing.
4. Verify Stripe configuration by reading `reports/beta_launch_ready_release.md`.
5. Do NOT modify backend limits or Stripe architecture without explicit permission. We are strictly in a Documentation & Launch preparation phase.
