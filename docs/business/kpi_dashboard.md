# KPI Dashboard Process

## Weekly KPI Review
Every Monday at 9:00 AM, the leadership team will review the following 5 Key Performance Indicators (KPIs) via the automated PostHog dashboard:

1. **Active Users (WAU):** Target > 2,000 WAU by Day 90.
2. **Investigation Volume:** Total scans processed across API and UI. Target > 50,000.
3. **Net MRR Churn:** Must remain < 3%.
4. **Pro → Enterprise Conversion Rate:** Target 5% conversion from self-serve to high-touch.
5. **Customer Support SLA Breaches:** Number of tickets that exceeded the 2-hour or 12-hour response windows. Target: 0.
