# Customer Success Process

## 1. Onboarding (First 7 Days)
- **High-Touch (Enterprise):** Dedicated CSM leads a 45-minute platform walkthrough. Focuses on setting up API keys, configuring the webhook, and adding team members.
- **Low-Touch (Pro):** Automated drip sequence (Days 1, 3, 5) guiding users to generate their first PDF report and extract their first watermark.

## 2. Retention (Ongoing)
- **Support Inbox:** Zendesk integration for API troubleshooting.
- **Predictive Interventions:** CS team reaches out manually to any Enterprise account whose API volume drops > 25% week-over-week.

## 3. Expansion (Quarterly)
- **QBRs:** A 30-minute review showcasing the total volume of deepfakes caught and evidence items secured on the blockchain. Use this time to negotiate seat expansions or higher API limits.
