# Support Plan (Launch Window)

## 1. Support Tiers
- **Tier 1 (Community):** Discord and Twitter community managers handling basic usage queries.
- **Tier 2 (Technical):** Dedicated support inbox (`support@fiduscan.com`) monitored by product engineers via Zendesk for API and integration issues.
- **Tier 3 (Enterprise):** Direct Slack Connect channels with enterprise design partners.

## 2. Coverage Schedule
During launch week (T-Zero to T-Plus 7), engineering will operate in rotating 12-hour shifts to ensure 24/7 coverage of Tier 2 and Tier 3 channels.

## 3. SLA Commitments
- **Enterprise:** < 2 hours response time.
- **Pro Tier:** < 12 hours response time.
- **Free Tier:** Best effort / 48 hours response time.
