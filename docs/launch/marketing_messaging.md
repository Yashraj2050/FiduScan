## Target Audiences

### 1. Trust & Safety Teams (Social Platforms / Dating Apps)
- **Pain Point:** Influx of deepfake accounts, catfishing, and synthetic spam causing user attrition.
- **Value Prop:** Sub-second API endpoints that integrate cleanly into existing moderation pipelines. Stop AI-generated content before it reaches the feed.
- **Key Feature:** Video temporal analysis & API throughput limits.

### 2. Journalists & News Organizations
- **Pain Point:** Verifying the authenticity of leaked audio recordings, warzone footage, or political images.
- **Value Prop:** Deep forensic explainability. We don't just output "Fake"; we provide Grad-CAM heatmaps and spectrogram analyses so you can visually prove the forgery.
- **Key Feature:** Web UI, Metadata Viewer, PDF reporting (coming soon).

### 3. Financial Services (KYC / Identity Verification)
- **Pain Point:** Synthetic ID fraud via deepfake injections bypassing facial recognition.
- **Value Prop:** Zero-retention privacy and bank-grade security. Cryptographic hashes ensure tamper-proof audit logs without storing sensitive customer faces.
- **Key Feature:** Zero-retention policy, Secure API keys.
