# FiduScan Referral Program

## Overview
A viral loop engineered to acquire new enterprise and pro users through trusted peer-to-peer recommendations.

## The Mechanism (Give $50, Get $50)
1. **Trigger:** A user completes 5 successful investigations (aha! moment).
2. **Action:** A modal prompts them to invite a colleague to try FiduScan.
3. **Reward:** 
   - **Inviter:** Receives $50 in API credits once the invitee converts to a paid plan.
   - **Invitee:** Receives a $50 discount on their first month of Pro.

## UI Integration
- Dedicated "Referrals" tab in the Settings navigation.
- Unique tracking links generated per user (e.g., `fiduscan.com/r/user_hash`).
- Dashboard widget showing pending and successful referrals with credit balance.
